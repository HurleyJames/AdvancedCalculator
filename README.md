## 简介
**高级计算器**是一款提供简单计算、汇率换算和亲戚关系计算的应用。

* 可执行加、减、乘、除等基础计算
* 可根据即时汇率换算所有的货币
* 可输入关系，得到亲戚的称呼
  
 
其中基础数字计算的算法采用的是中缀转后缀表达式的算法。
